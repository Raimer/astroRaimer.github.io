## Markdown Note

### Text style

Use ```font```, e.g.:

```
<font color=red size=16> Hello World </font>
```

Output:

<font color=red size=16> Hello World </font>

Use ```span style```, e.g.:

``` 
<span style=" background-color: green; color:white; font-weight:800; font-size:25px">Hello World</span>
```

Output:

<span style=" background-color: green; color:white; font-weight:800; font-size:25px">Hello World</span>
